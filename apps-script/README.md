# Trophy Auto-Awarding (Apps Script)

Скрипт `trophies-auto.gs` автоматически записывает строки в лист
`TrophyAwards` в основной таблице дашборда. Фронт уже умеет читать
этот лист и показывать карточки в разделе «Трофеи».

## Что делает скрипт

Три точки входа:

| Функция | Когда запускать | Что делает |
|---|---|---|
| `awardMonthlyTrophies()` | 2-го числа каждого месяца, например в 03:00 | Считает итоги ПРЕДЫДУЩЕГО месяца (ВИЗИТЫ + Д_ВИЗИТЫ + ПЛАН) и пишет AUTO-трофеи |
| `awardDailyTrophies()` | Каждый день, например в 09:00 | Раздаёт `hb_annual` тем, у кого сегодня ДР |
| `awardHnyTrophies()` | 1 января, ~00:05 | Массово раздаёт `hny_annual` всем менеджерам |

## Установка

1. Открой основную таблицу дашборда в Google Sheets.
2. **Extensions → Apps Script**. Откроется редактор.
3. В дереве файлов создай новый файл `trophies-auto.gs` (или замени
   `Code.gs`) и вставь содержимое из репо.
4. Сверху проверь `SHEET_ID` — должен быть ID **этой же таблицы**
   (по умолчанию подставлен `1DeUsHB...` — это дашборд).
5. Сохрани (Cmd/Ctrl+S).

## Установка триггеров

Слева в редакторе → значок **«Triggers»** (часы) → **+ Add Trigger** (правый нижний угол).

### Триггер 1 — ежемесячный

- Function: `awardMonthlyTrophies`
- Event source: **Time-driven**
- Type: **Month timer**
- Day of month: **2**
- Time: **3am – 4am** (любое раннее время)
- Failure notification: **Notify me daily**

### Триггер 2 — дневной (ДР)

- Function: `awardDailyTrophies`
- Event source: **Time-driven**
- Type: **Day timer**
- Time: **9am – 10am**

### Триггер 3 — Новый год

- Function: `awardHnyTrophies`
- Event source: **Time-driven**
- Type: **Specific date and time** или **Year timer**
  *(Apps Script не имеет годового триггера в UI — лучше поставить
  «Day timer» и оставить логику самому скрипту: ничего не пишет,
  если не 1 января)*

Альтернатива для №3: ставишь Day-timer и добавляешь в начало
`awardHnyTrophies`:
```js
const d = new Date();
if (!(d.getMonth() === 0 && d.getDate() === 1)) return;
```

## Первый запуск

1. В редакторе выбери в шапке функцию **`debugLastMonth`** → нажми ▶
   **Run**. Google попросит разрешения — выдай (Sheets API).
2. Глянь логи (View → Logs). Должно быть `[trophies] YYYY-MM: добавлено
   выдач N`.
3. Открой лист **TrophyAwards** — там появятся свежие строки.
4. Открой фронт-дашборд → раздел «Трофеи» → проверь, что карточки
   окрасились как полученные.

Если что-то не понравилось — удали добавленные строки прямо в листе
и поправь логику в скрипте. Скрипт **идемпотентен**: повторный запуск
не создаст дубли (проверяется по `code + manager + period`).

## Какие трофеи раздаёт сейчас

**AUTO ежемесячные** (всё считается из ВИЗИТЫ + Д_ВИЗИТЫ + ПЛАН):

- `numberone/two/three_monthly` — топ-3 по % выполнения плана (≥100%)
- `star_monthly` (AWESOME) — план ≥130% + 30 кредитных продаж
- `star2_monthly` (NICE) — план ≥120% + 20 кредитных продаж
- `commission_monthly` (Коллектор) — max сделок «Комиссия»
- `kd_monthly` (Счастливое число CRM) — 33+ кредита за месяц
- `cash_monthly` (Туз в рукаве) — 10+ продаж за наличные
- `3in1_monthly` (Три в ряд) — 3 кредита в один день
- `notfailure_monthly` (Идеальная неделя) — 7 дней подряд без отказов/ФССП

**AUTO ежегодные**:

- `hb_annual` — Happy Birthday (в день рождения из USERS!F)
- `hny_annual` — HNY (1 января всем)

## Структура `TrophyAwards`

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| `trophy_code` | `manager_name` | `awarded_at` | `awarded_by` | `source` | `status` | `note` |
| `numberone_monthly` | `Бочаров Юлиан` | `2026-06-02` | `system` | `auto` | `active` | `2026-05 · 142%` |

- **status=locked** — трофей залочен от автомата (если ты его
  override’ил руками и не хочешь чтобы AUTO его сносил). Скрипт
  такие пары `code|manager` не трогает.
- **note** содержит период (YYYY-MM для месячных, YYYY для годовых)
  + полезный контекст (процент, кол-во сделок). Это нужно для
  идемпотентности — повторный запуск проверяет по подстроке периода.

## Логи и отладка

- View → Logs (новые) или View → Executions (история запусков).
- Если триггер упал — в почту прилетит письмо «Apps Script:
  Daily/Monthly trigger failed».
- Ручной перепрогон конкретного месяца:
  ```js
  rerunFor('2026-05')
  ```
  Запусти эту функцию в редакторе.
