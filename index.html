Режим сверки

При активации система использует только визиты со значением Да в столбце Сверка для сверочных расчётов и дохода. Отображаемые факты визитов, продажи, KPI, рейтинг и журнал визитов остаются по всем внесённым данным.

Режим вставки визитов

При активации система включает возможность вставки визита между другими визитами через кнопку +. Деактивация выключает этот режим и остаётся возможность добавлять новые визиты только через кнопку Добавить визит.

Техническое обслуживание

При активации система отображает экран технического обслуживания для пользователей CRM и Дожим. Отец основатель и Лида работают в штатном режиме.

О проекте
CRM Crew Dashboard / CM66

Внутренний дашборд для команды Crystal Motors — сети автосалонов с 12 городами присутствия. Создан как замена ручным отчётам и разрозненным таблицам.

Цель

Дать каждому менеджеру и руководителю единое окно для мониторинга KPI, динамики продаж, рейтингов, зарплаты и расписания — в реальном времени, с любого устройства.

Возможности
📊 KPI — визиты, кредиты, наличные, комиссии по каждому менеджеру
🏆 Рейтинг — топ менеджеров с анимированными барами и медалями
💰 Доход — расчёт зарплаты с коэффициентами и котлом
📅 График — расписание смен, просмотр по неделям
📋 Визиты — онлайн-журнал с автосохранением в Google Sheets
🎂 Дни рождения — уведомления о праздниках коллег
🎨 Темы — Glass, Tiffany, Cinematic, Neo, Cosmic
📱 PWA — работает как приложение на iOS и Android
Стек

Single-file HTML · Google Sheets API v4 · Google OAuth2 · Three.js · WebGL GLSL
